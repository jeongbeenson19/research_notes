
# Corpus (말뭉치)

#NLP #Concept #Dataset

---

## 1. 개요

**말뭉치(Corpus)** 또는 **코퍼스**는 자연어 처리(NLP) 및 언어학 연구를 위해 특정 목적을 가지고 수집된 대규모의 구조화된 텍스트 데이터 집합을 의미합니다. 이는 언어의 실제 사용 용례를 보여주는 원시 데이터로서, 통계적 분석, 언어 모델 학습, 가설 검증 등 다양한 작업의 기반이 됩니다.

NLP 모델에게 말뭉치는 인간의 언어를 학습하고 이해하기 위한 '교과서'와 같은 역할을 합니다. 모델은 말뭉치에 나타난 단어의 분포, 문장 구조, 문맥적 의미 등을 학습하여 특정 과업(Task)을 수행하는 능력을 기릅니다.

---

## 2. 말뭉치의 종류

말뭉치는 구성 언어, 구조, 주석(Annotation) 여부 등에 따라 다양하게 분류될 수 있습니다.

### 2.1. 구성 언어에 따른 분류

-   **단일 언어 말뭉치 (Monolingual Corpus):** 하나의 언어로만 구성된 말뭉치입니다. 특정 언어의 내부적인 특징(문법, 의미 등)을 분석하거나 해당 언어를 사용하는 언어 모델을 훈련시키는 데 사용됩니다.
-   **다중 언어 말뭉치 (Multilingual Corpus):** 둘 이상의 언어로 구성된 말뭉치입니다.
    -   **병렬 말뭉치 (Parallel Corpus):** 동일한 내용의 텍스트가 여러 언어로 번역되어 쌍을 이루고 있는 말뭉치입니다. (예: "Hello" - "안녕하세요"). 주로 기계 번역 모델 훈련에 필수적으로 사용됩니다.
    -   **비교 말뭉치 (Comparable Corpus):** 번역 관계는 아니지만, 각 언어에서 동일한 주제, 장르, 시기에 작성된 텍스트를 모아놓은 말뭉치입니다. 여러 언어 간의 문화적 차이나 스타일을 비교 연구하는 데 사용됩니다.

### 2.2. 주석(Annotation) 여부에 따른 분류

-   **원시 말뭉치 (Raw Corpus):** 텍스트 원문만 정제하여 모아놓은 말뭉치입니다.
-   **주석 말뭉치 (Annotated Corpus):** 언어학적 정보를 태그(Tag) 형태로 추가한 말뭉치입니다. 시간과 비용이 많이 들지만, 특정 NLP 작업의 성능을 높이는 데 결정적인 역할을 합니다.
    -   **형태소 분석 말뭉치:** 각 단어의 품사(POS)와 형태소 정보를 태깅합니다. (예: `아버지/NNG`, `가/JKS`)
    -   **구문 분석 말뭉치 (Treebank):** 문장의 구성 성분과 문법적 구조를 트리(Tree) 형태로 표기합니다. (예: Penn Treebank)
    -   **의미 분석 말뭉치:** 단어의 중의성을 해소하거나 개체명(Named Entity) 등을 표기합니다.

## 3. 말뭉치의 구축과 전처리

좋은 품질의 말뭉치를 구축하는 것은 성공적인 NLP 프로젝트의 핵심 요소입니다.

1.  **데이터 수집 (Data Collection):** 웹 크롤링, 뉴스 기사, 도서, 소셜 미디어 등 다양한 소스에서 텍스트를 수집합니다.
2.  **정제 (Cleaning):** HTML 태그, 특수 문자, 중복 데이터 등 불필요한 노이즈를 제거합니다.
3.  **전처리 (Preprocessing):** 수집된 데이터를 모델이 학습할 수 있는 형태로 가공합니다. 이 과정은 말뭉치를 어떻게 활용할지에 따라 달라집니다.
    -   **[[../../../10_Concepts/Machine_Learning/Tokenization|토큰화 (Tokenization)]]:** 문장을 의미 있는 단위(토큰)로 분리합니다. ([[Python Tokenizers|Python Tokenizers]] 참고)
    -   **정규화 (Normalization):** 대소문자 통일, 문장 부호 처리, 불용어(Stopword) 제거 등을 수행합니다.
    -   **주석 작업 (Annotation):** 품사 태깅, 개체명 인식(NER), 구문 분석 등의 언어학적 정보를 추가합니다.

## 4. 주요 말뭉치 예시

-   **영어권**
    -   **Brown Corpus:** 다양한 장르의 텍스트를 균형 있게 포함한 최초의 현대적 말뭉치 중 하나입니다.
    -   **Penn Treebank:** 문장 구조가 태깅된 구문 분석 말뭉치로, 많은 구문 분석 연구의 기반이 되었습니다.
    -   **British National Corpus (BNC):** 영국 영어를 대표하는 대규모 말뭉치입니다.
-   **한국어**
    -   **세종 말뭉치:** 국립국어원에서 구축한 한국어 표준 말뭉치로, 원시 말뭉치와 다양한 주석 말뭉치를 포함합니다.
    -   **모두의 말뭉치 (MODU):** 국립국어원에서 공개한 대규모 한국어 말뭉치로, 뉴스, 대화, 웹 문서 등 다양한 출처의 데이터를 포함합니다.

## 5. NLP에서의 활용

-   **언어 모델 훈련:** [[Word2Vec]], [[GloVe]], [[BERT]], [[GPT]]와 같은 언어 모델은 대규모 원시 말뭉치를 통해 단어와 문장의 의미적 표현을 학습합니다.
-   **기계 번역:** 병렬 말뭉치는 번역 모델이 출발어와 도착어 간의 변환 규칙을 학습하는 데 사용됩니다.
-   **품사 태깅 및 개체명 인식:** 주석 말뭉치는 특정 단어가 문맥 속에서 어떤 품사로 쓰이는지, 또는 '인물', '기관' 등 어떤 개체명에 해당하는지를 분류하는 모델을 훈련시킵니다.
-   **감성 분석:** 리뷰나 소셜 미디어 텍스트에 긍정/부정 레이블을 단 말뭉치를 통해 텍스트의 감성을 예측하는 모델을 학습합니다.
