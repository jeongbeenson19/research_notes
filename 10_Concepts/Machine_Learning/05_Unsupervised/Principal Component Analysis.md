# Principal Component Analysis

## 한줄 요약
분산이 큰 방향을 찾아 차원을 축소하는 선형 변환 기법.

## 핵심 개념
- 공분산 행렬의 고유벡터/고유값 기반
- 정보 손실을 최소화하며 저차원 표현
- 시각화와 전처리에 널리 사용

## 대표 수식
- 공분산 행렬의 고유분해 또는 SVD

## 예시
- 고차원 특징을 2D로 투영해 시각화

## 실무 팁/주의점
- 입력을 표준화한 뒤 적용하는 것이 일반적
- 선형 관계만 모델링 가능
