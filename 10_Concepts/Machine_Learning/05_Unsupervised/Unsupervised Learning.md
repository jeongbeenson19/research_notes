# Unsupervised Learning

**Unsupervised Learning(비지도 학습)**은 머신러닝의 한 패러다임으로, 정답 레이블(label)이 없는 데이터를 사용하여 데이터 자체의 숨겨진 구조, 패턴, 관계를 학습하는 방법입니다.

모델은 데이터의 내재적 특성을 스스로 파악하여 그룹화하거나, 데이터의 차원을 축소하고, 확률 분포를 추정하는 등의 작업을 수행합니다.

## 주요 유형

-   **[[Clustering|클러스터링]]**: 유사한 데이터 포인트들을 그룹으로 묶는 작업. (예: K-Means, DBSCAN)
-   **[[Dimensionality Reduction|차원 축소]]**: 데이터의 주요 특징을 유지하면서 데이터의 차원(feature의 수)을 줄이는 작업. (예: PCA, t-SNE)
-   **연관 규칙 학습 (Association Rule Learning)**: 데이터 항목 간의 흥미로운 관계를 발견하는 작업. (예: Apriori)

[[Hebbian Rule|헤비안 규칙]]은 대표적인 비지도 학습 원리 중 하나입니다.
