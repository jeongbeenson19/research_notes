# Adaptive Resonance Theory (ART)

**Adaptive Resonance Theory (ART)**, 즉 적응 공명 이론은 **Stephen Grossberg**와 **Gail Carpenter**가 제안한 신경망 이론으로, 인간의 지각(perception) 및 학습 과정을 설명하려는 모델입니다.

이 이론의 핵심은 **새로운 정보에 유연하게 적응(plasticity)하면서도 기존의 학습된 지식을 안정적으로 유지(stability)하는 능력**을 신경망으로 구현하는 데 있습니다.

> **요약**: ART는 입력 패턴을 기존 범주와 비교해 공명 여부를 판단하며, 새 범주를 생성하거나 기존 범주를 업데이트하는 방식으로 **안정성과 유연성을 동시에 추구하는 신경망 학습 이론**입니다.

## 📖 배경: Stability-Plasticity Dilemma

전통적인 신경망(예: Hopfield network, Perceptron)은 새로운 정보를 학습할 때 **기존에 학습된 패턴을 잊어버리는 문제(catastrophic forgetting)** 가 있습니다. ART는 이 '안정성-유연성 딜레마'를 해결하는 것을 주요 목표로 합니다.

## ⚙️ 핵심 구성 요소 및 원리

ART는 크게 두 개의 층(Layer)과 상호작용 메커니즘으로 구성됩니다.

-   **입력층 (F1 Layer)**: 입력 신호를 처리하고 현재 자극을 표현합니다.
-   **출력층 (F2 Layer)**: 입력 패턴을 분류하는 범주(category)를 나타냅니다.
-   **Vigilance Parameter (경계 변수)**: 입력 패턴이 기존 범주와 얼마나 유사해야 '같은 범주'로 판단할지를 결정하는 임계값입니다. 이 값을 통해 범주의 세분화 수준을 조절할 수 있습니다.

### 작동 원리

1.  입력 패턴이 F1 층으로 들어옵니다.
2.  F1은 F2 층의 모든 범주 뉴런과 유사도를 비교하여, 가장 유사한 범주 뉴런을 잠정적으로 선택합니다.
3.  선택된 범주의 패턴과 입력 패턴의 유사도가 **Vigilance Parameter** 값보다 크면 **"공명(Resonance)"**이 발생하고, 해당 범주의 가중치가 입력 패턴에 더 가깝게 업데이트됩니다 (학습 진행).
4.  유사도가 기준에 미치지 못하면 **"불일치(Mismatch)"**가 발생하고, 선택됐던 뉴런은 억제된 후 다른 범주를 탐색합니다. 만약 더 이상 일치하는 범주가 없으면, 새로운 범주를 생성합니다.

## ✨ 주요 특징

| 특징                            | 설명                                                 |
| ------------------------------- | ---------------------------------------------------- |
| **Stability-Plasticity 해결**   | 기존 지식을 잊지 않으면서 새로운 정보를 학습합니다.  |
| **비지도 및 준지도 학습**       | 라벨이 없는 데이터도 스스로 분류(클러스터링) 가능합니다. |
| **범주 수 자동 결정**           | 입력 데이터에 따라 동적으로 새로운 범주를 생성합니다.    |
| **온라인 학습 (Online Learning)** | 데이터가 들어오는 즉시 실시간으로 학습할 수 있습니다.    |

## 🧩 ART의 변형

-   **ART1**: 이진(binary) 입력만 처리합니다.
-   **ART2**: 연속적인(analog) 아날로그 입력을 처리할 수 있습니다.
-   **ART3**: 시냅스 전달과 같은 생물학적 프로세스를 더 현실적으로 모델링합니다.
-   **Fuzzy ART**: 퍼지 논리(Fuzzy Logic)를 통합하여, 각 입력이 여러 범주에 속하는 정도를 표현할 수 있습니다.
