# Trend 정의(1문장)
- 분~시간 단위의 장기 비디오를 이해·질의응답하는 연구 흐름을 의미한다.

이 정의를 바탕으로, 왜 어려운지 핵심 요인을 정리한다.

## 왜 어려운가(3)
- 입력 길이/토큰 예산 제약으로 전체 컨텍스트 보존이 어렵다.
- 장시간에서 사건·행동의 장기 의존성을 추론하기 어렵다.
- 라벨링 비용과 평가 데이터 다양성 부족으로 일반화가 어렵다.

이 어려움을 해결하기 위해 대표적인 접근 유형을 분류해 본다.

## 대표 접근 유형(2~3)
- Type A: 메모리·요약 기반 장기 컨텍스트 유지(예: memory, compression).
- Type B: 계층적 처리(클립-세그먼트-에피소드 수준).
- Type C: 검색/회수 기반 외부 메모리 결합.
